# Portfolio personal

Este es mi portafolio web, hecho con Astro.
Aquí muestro mis proyectos principales (Higinex y Pleroma), un resumen sobre mí y formas de contacto.

## Stack

- Astro
- Tailwind CSS

## Ejecutar en local

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```
